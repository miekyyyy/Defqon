# Defqon Stage

Whats cooler than having your own mini stage at home?!

---

## Overview

This project is designed to be used together with the [Defqon Editor](https://github.com/miekyyyy/DefqonUnity).

---

## Features

* 🔹 Multiple effect types
* 🔹 SFTP file transfering, so the stage can be used offline.

---

## Tech Stack

List the main tools and technologies used:

* **Frontend:** Editor (C#, Unity)
* **Backend:** Raspberry PI (C#), Esp32 linked for lighting (C++)

## ⚙️ Usage

1. Design a light show in the editor
2. Upload it to the stage
3. Play
4. Enjoy!
